# Codex Shelf

Codex Shelf is a full-stack book application where users can browse books, publish their own books, and leave reviews.

## Features

- User sign up, sign in, and sign out
- View all books
- View book details
- Add new books
- Edit and delete books
- Add and delete reviews
- Authorization for books and reviews
- Responsive mobile design
- Custom 404 page
- Delete confirmation
- Empty state when there are no books
- Password validation

## Technologies Used

- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- CSS
- bcrypt
- express-session
- method-override
